# Proyecto Severo — Relevamientos

## Design Context

This project uses **impeccable** design context. Before any UI work, read **`PRODUCT.md`** (and `DESIGN.md` once generated) at the repo root.

- **Register:** `product` (field tool; design serves the relevamiento task, not a marketing surface).
- **Users:** mixed-skill grassroots volunteers canvassing door to door on their own phones, outdoors, one-handed, on unreliable connections.
- **Core job:** fast, accurate citizen capture against the padrón in under a minute.
- **Strategic principles:** Speed at the door · Trust through legibility, not chrome · Never lose the volunteer's work · Field-first ergonomics · Dignified, not surveillant.
- **Strongest anti-reference:** the clunky government portal (dated, bureaucratic, gray dense forms).

Run `/impeccable <command>` for design work; it loads PRODUCT.md/DESIGN.md automatically.
